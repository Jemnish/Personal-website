Farmerket

jenishbudhathoki.com.np > Farmerket

Farmerket is a secure MERN-based web application built with advanced security practices to protect user data and ensure a safe user experience. By following OWASP guidelines, the platform mitigates common web vulnerabilities while strengthening authentication, authorization, and data protection mechanisms. Key features include password hashing with bcrypt.js, JWT-based session management, multi-factor authentication (MFA), input validation with sanitize-html, CSRF protection, and rate limiting. The platform also implements HTTPS with SSL for encrypted communication, role-based access control (RBAC) in MongoDB, and bot protection through Google reCAPTCHA, making Farmerket a highly secure and user-friendly solution.
